# ERMA Website

Static sites for erma.ro and sere.erma.ro.

## Structure

- `catalog-site/` — main catalog site at erma.ro
- `sere-site/` — greenhouse business site at sere.erma.ro
- `docs/` — project documentation

## Editing catalogs

Catalog content lives in a Google Sheet:
https://docs.google.com/spreadsheets/d/[...]/edit

The site rebuilds on push to `main`. Edits to the sheet take effect on the next rebuild.
